# Daily Summary

오늘 작업한 내역을 확인합니다.

## 실행 방법

1. `~/.claude/daily-work/` 디렉토리에서 오늘 날짜 파일을 읽으세요
2. 파일 형식: `YYYY-MM-DD.md` (예: 2026-01-05.md)
3. 파일이 없으면 "오늘 기록된 작업이 없습니다" 출력

## 출력 형식

파일 내용을 보여주고 아래 통계 추가:

```
---
📊 오늘 통계
- 작업한 프로젝트: N개
- 총 대화 횟수: N회
```

## 인자

- 날짜 지정 가능: `/daily-summary 2026-01-04` → 해당 날짜 파일 조회
- 인자 없으면 오늘 날짜
