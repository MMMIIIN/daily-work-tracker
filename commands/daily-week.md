# Daily Week

이번 주 작업 내역을 요약합니다.

## 실행 방법

1. `~/.claude/daily-work/` 디렉토리에서 최근 7일간의 파일들을 읽으세요
2. 각 날짜별로 프로젝트와 주요 작업 요약
3. 전체 통계 제공

```markdown
## 2026-01-07 (화)
- **flutter-app**: 사용자 인증 구현 (5회)

- **flutter-app**: UI 작업 (4회)

---
📊 주간 통계
- 총 작업일: 2일
- 작업한 프로젝트: 2개
- 총 대화 횟수: 12회
```
